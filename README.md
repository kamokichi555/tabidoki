# 旅刻 (tabidoki) — ツーリング行程支援ツール

> バイクツーリングの行程管理に特化した、ブラウザで動くWebアプリです。

**👉 [アプリを開く](https://kamokichi555.github.io/tabidoki/)**

---

## こんな人に

- ツーリングの行程をスマホで管理したい
- 出発前に天気・給油ポイント・SA/道の駅をまとめて確認したい
- 走りながら現在地点をすぐ確認したい

---

## 主な機能

| 機能 | 説明 |
|---|---|
| 🗺 行程作成 | 地点・到着/出発時刻・メモを登録。複数日程に対応 |
| ⏱ 時刻の自動ずれ | 出発時刻を変えると以降の地点が自動でずれる |
| 🌤 天気取得 | 各地点の天気予報を自動取得（住所入力不要） |
| 🛣 施設ピッカー | 高速SA/PA・道の駅・GSをマップ検索なしで選択 |
| 🏍 走行モード | 現在地点を大きく表示。GPS自動追従対応 |
| 📡 GPS追従 | 走行中に現在地に最も近い地点へ自動で切り替え |
| 💾 JSON保存・共有 | 行程をJSONファイルで保存・読み込み・テキスト共有 |
| 📴 オフライン対応 | PWA。機内モードでも動作（ホーム画面追加推奨） |

---

## 使い方

### 1. 行程を作る（編集モード）

1. ✏️ ボタンで編集モードを開く
2. 地点名を入力 → 高速・道の駅・GSはピッカーで選べる
3. 到着・出発時刻・メモを入力して「追加」
4. 地点は長押しドラッグで並べ替え可能

### 2. 走行中（走行モード）

1. 🏍️ ボタンで走行モードに切り替え
2. 現在地点が大きく表示される
3. 📡 ボタンでGPS自動追従をON → 走りながら地点が自動で切り替わる

### 3. 保存・共有

- 💾 でJSONファイルに保存
- 📋 でテキスト形式に変換してコピー・共有
- 📂 で保存済みJSONを読み込む

---

## ホーム画面への追加（推奨）

オフラインで動かすにはPWAとしてホーム画面に追加してください。

- **iOS Safari**：共有ボタン →「ホーム画面に追加」
- **Android Chrome**：メニュー →「アプリをインストール」
