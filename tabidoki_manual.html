Chapter 0

使用要件・推奨環境

0-1 アクセス方法

ブラウザで開くだけで使えます

https://kamokichi555.github.io/tabidoki/

インストール不要。上記URLをブラウザで開くだけで動作します。単一のHTMLファイルで動いているため、ページを読み込んだ後はオフラインでも基本機能が使えます。

ホーム画面に追加すると便利
スマートフォンのブラウザから「ホーム画面に追加」すると、アプリのように起動できます。Android は Chrome、iOS は Safari から追加してください。
0-2 推奨ブラウザ・デバイス
🤖
Android Chrome
最推奨

動作確認済み。ファイル保存・天気取得すべて正常動作。

🍎
iOS Safari
推奨

動作確認済み。ファイル保存は新しいタブで開いて長押し保存の操作になります。

💻
PC Chrome / Edge
動作可

PCでも動作しますが、スマートフォン向けに設計されています。

🦊
Firefox
動作可

基本機能は動作します。一部の表示が異なる場合があります。

非推奨：Internet Explorer はサポート対象外です。旧バージョンのブラウザでは正しく動作しない場合があります。
0-3 通信要件
🌐

ネット接続初回のみ必須

ページの読み込みにはインターネット接続が必要です。読み込み後はオフラインでも行程の作成・編集・閲覧ができます。

🌤️

天気予報の取得通信時のみ有効

住所が設定された地点の天気予報は、オンライン時に自動取得されます。オフラインでは天気が表示されません。

🏪

SA・PA・道の駅の検索通信推奨

オンライン時は最新データをリアルタイム取得します。オフライン時は内蔵データで動作しますが、情報が古い場合があります。

オフラインでできること
行程の作成・編集・閲覧・時刻管理・走行モード・区間時間の計算・記録の入力・テキスト保存はすべてオフラインで使用できます。
0-4 ストレージ・データ保存

localStorage への保存

行程データはブラウザの localStorage に自動保存されます。容量上限はブラウザによって異なりますが、一般的に 5〜10MB です。通常の使用では上限に達することはありません。

💾 JSONバックアップの推奨

ブラウザのデータ消去・機種変更・再インストール時にlocalStorageのデータが消えます。大切な行程は必ず 💾 ボタン で JSON ファイルとして書き出しておいてください。

シークレットモード・プライベートブラウズは非推奨
これらのモードではlocalStorageへの保存ができないか、ブラウザ終了時にデータが消去されます。通常モードでご利用ください。

✏️ 編集モードとは

ヘッダーの ✏️ ボタンをタップして入る画面。地点の追加・編集・並び替え・削除を行います。走行モードへ切り替えるか ✅ を押すと編集モードが終了します。

2-1 日程タブ

画面上部に日程タブが表示されます。現在選択中の日程は 1日目 のようにamber色でハイライト。 最大 7日間 まで追加できます。 日程をスワイプして切り替え可能です。

＋

日程追加

新しい日（2日目・3日目…）を追加します。前日の日付が設定されていれば翌日の日付が自動入力されます。

🗑️

削除

現在表示中の日程を丸ごと削除します。地点が1件もない場合は全クリアになります。

📅

日付入力

ツーリングの日付を設定します。設定すると天気予報が取得できます。日付順に自動でタブが並び替わります。

🗺️

ルートURL

Touring Supporter などのルート共有URLを貼り付けると、走行モードから直接開けます。

2-2 地点の追加フォーム
フォーム上部
フォーム下部
地点名必須

立ち寄り先の名前

「道の駅みつまた」「宮ヶ瀬湖」など。最大50文字。

高速 / 道の駅 / GS

入力補助ボタン

タップすると候補リストが開き、選ぶだけで地点名・住所を自動入力できます。高速はSA・PA・IC、道の駅は全国対応、GSはチェーン選択＋給油ポイント自動ON。

着 / 発任意

到着・出発時刻

どちらか一方だけでもOK。両方入力すると滞在時間が自動計算されます。出発時刻を変更すると以降の地点の時刻が自動でずれます（カスケード）。

メモ任意

計画メモ

「名物の鹿カレー」「保土ヶ谷バイパスを避けて下道で」など、次回以降の計画に役立つ情報。走行モードでも表示されます。

帰宅後に「飯がうまかった」「路面が荒れていた」などを記録。計画メモと別管理で、青い左ボーダーで区別表示されます。📝 記録保存ボタンの点灯条件にもなります。

行程テキスト共有

共有画面

3-1 この機能について

📋 行程テキストとは

登録した行程を人間が読みやすいテキスト形式に変換して表示します。ヘッダーの 📋 ボタン をタップすると表示されます。

💡 LINE・メモ・メール・メモアプリへの貼り付けに使えます。行程の共有やバックアップとして活用してください。

3-2 テキストの内容

変換されるテキストには以下の情報が含まれます。

【1日目】 5/23(金)

▶ 横浜 出発
　出発 08:00
　📝 早めに出発。首都高・保土ヶ谷バイパスを避けて下道で

▶ 宮ヶ瀬湖
　到着 09:30 ｜ 出発 10:00
📅

日程ごとに区切り表示

複数日のツーリングは【1日目】【2日目】…と日程ごとに整理されます。日付設定がある場合は日付も表示。

▶

地点名と時刻

地点名の前に ▶ が付き、到着・出発時刻が「到着 09:30 ｜ 出発 10:00」の形式で表示されます。

📝

メモ・走行メモ

各地点に登録したメモ（計画メモ・走行メモ）がそのまま含まれます。

3-3 操作方法
1

📋 ボタンをタップ

ヘッダーのクリップボードボタンをタップします。行程テキスト画面が開きます。

2

内容を確認

スクロールして行程テキストの内容を確認できます。

3

📋 クリップボードにコピー をタップ

ボタンをタップするとテキスト全体がクリップボードにコピーされます。「✅ コピーしました」と表示されたら完了。画面は自動で閉じます。

4

貼り付けて使う

LINE・メモ・メール等のアプリで貼り付けます。仲間への行程共有やバックアップとして活用できます。

× ボタンで閉じる
コピーせずに閉じたい場合は右上の × をタップします。背景部分をタップしても閉じられます。

📜 設定画面を開くには

ヘッダーの 📜 ボタン（巻物アイコン）をタップします。スプラッシュ演出のカスタマイズと、デバッグログの管理ができます。

設定後は ✅ 保存して閉じる をタップして確定。× ボタンや背景タップでキャンセルできます。

4-1 スプラッシュ演出の設定

アプリ起動時に表示される映画風スプラッシュ画面の演出を自分好みにカスタマイズできます。

主演名

あなたの名前

スプラッシュ画面に「主演：〇〇」として表示されます。空欄にすると「あなた」が入ります。例：田中太郎

第○話

話数（数字で入力→漢数字に変換）

数字を入力するとリアルタイムで漢数字に変換されます。「12」と入力すると 第十二話 と表示。空欄にするとランダムな話数が設定されます。

タイトル

サブタイトル（紋次郎風ランダム）

スプラッシュ画面に表示されるサブタイトル。空欄にすると「甲州街道に月を追う」のような紋次郎風のタイトルがランダムで生成されます。例：甲州街道に月を追う

設定するとこのように表示されます：
第十二話
旅刻
主演：田中太郎
甲州街道に月を追う
4-2 デバッグログ

不具合が発生したときに操作履歴・エラー情報を記録・取り出す機能です。通常は OFF のままで問題ありません。

☑️

操作・エラーを記録する（ON/OFF）

チェックするとログ記録を開始します。ONにすると画面右下に 🐛 バッジが表示されます。記録件数とエラー件数がリアルタイムで確認できます。

📋

コピー

記録されたログをクリップボードにコピーします。不具合報告の際にそのまま貼り付けて送れます。

💾

保存

ログをテキストファイルとして保存します。

🗑️

消去

記録されたログをすべて削除します。

不具合が起きたときの手順
① 「操作・エラーを記録する」をON
② 不具合を再現する操作を行う
③「コピー」または「保存」でログを取り出す
④ 報告先にログを貼り付けて送る
注意：デバッグログをONにしたまま長時間使い続けると、ログが大量に蓄積されることがあります。不具合の確認が終わったらOFFにするか「消去」してください。

5-1 著作権

本ツールの著作権

旅刻mk12（以下「本ツール」）のソースコード・デザイン・UIを含む一切の著作権は、制作者（鴨吉）に帰属します。

本ツールは個人利用を目的として制作されたものです。無断転載・再配布・商用利用・改変した上での公開はご遠慮ください。

利用データの著作権

本ツールに入力した行程データ（地点名・メモ・時刻・走行記録等）の著作権は、入力したご本人に帰属します。本ツールがこれらのデータを外部に送信・収集することはありません。

外部サービスの利用

本ツールは以下の外部サービスを利用しています。各サービスの著作権および利用規約はそれぞれの提供元に帰属します。

• Open-Meteo　天気予報データの取得
• OpenStreetMap Nominatim　住所→座標変換
• 国土地理院 ジオコーダAPI　住所→座標変換（補完）
• OpenStreetMap Overpass API　SA・PA・道の駅データ取得
• Google Maps　マップリンク表示
• Google Fonts　フォントデータ（Yuji Boku 等）

⚠ 安全運転に関する免責

本ツールは走行計画の補助を目的としたものです。走行中の操作は絶対に行わないでください。

本ツールの使用に起因する事故・違反・損害について、制作者は一切の責任を負いません。交通ルールを遵守し、安全運転に努めてください。

⚠ データ損失に関する免責

本ツールのデータはブラウザの localStorage に保存されます。以下の操作によりデータが消失する場合があります。

• ブラウザの「閲覧データの消去」
• プライベートブラウズ（シークレットモード）の終了
• ブラウザのアンインストール・再インストール
• ブラウザのストレージ上限超過

大切な行程データは必ず 💾 保存ボタン で JSON ファイルとしてバックアップを取ってください。

天気予報・外部データに関する免責

天気予報・施設情報（SA・PA・道の駅等）は外部サービスのデータを利用しています。情報の正確性・最新性を保証するものではなく、これらの情報に基づく判断で生じた損害について制作者は責任を負いません。

サービス継続性に関する免責
