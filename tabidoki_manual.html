FIRST PAGE

01 ・ 動作環境

旅刻はインストール不要のWebアプリです。ブラウザでURLを開くだけで使えます。

推奨ブラウザ
Android Chrome / iOS Safari最新版を推奨。Firefoxでも動作します。
通信
必須（オンライン専用）天気予報・住所検索・地図リンクで通信が発生します。
位置情報
任意（許可推奨）「近くのトイレ」「快活CLUB検索」で使用。
データ保存
端末のlocalStorageJSONファイルでバックアップ・引き継ぎ可。
配信
GitHub Pageshttps://kamokichi555.github.io/tabidoki/
料金
無料・広告なしサーバー維持費はゼロ円運用。
注意： ダウンロードしたHTMLファイルを直接開く（file://）と、位置情報・天気APIなど一部機能が動作しません。必ずWebアドレス経由でアクセスしてください。

11 ・ データの保存と読込

作った行程は端末に自動保存され、JSONファイルとしてバックアップもできます。

フォルダに保存されます
💾JSONとして書き出し
📂JSONから読み込み
🔄自動保存（localStorage）
2種類の保存
  • 自動保存：操作するたびlocalStorageへ
  • JSON保存：ファイルでバックアップ
  • 機種変・他端末への引継ぎに使う
  • iOS Safariは別タブで保存指示
注意： ブラウザのキャッシュをクリアすると自動保存データが消えます。大切な行程は必ず💾でJSONバックアップを取ってください。
